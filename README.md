# LeongLauncher


###简介
这个LeongLauncher 初始是基于原生8.1 的Launcher3版本开发的无抽屉Launcher，也就是类似小米Launcher或者iphone Launcher 这种所有应用的图标都排在桌面上，不再有原生Launcer抽屉Allapp的界面。

如果想要查看原生Launcher8.1的代码可以切换到分支aosp_8.1上，方便查看原生代码。
```
git checkout aosp_8.1
```
